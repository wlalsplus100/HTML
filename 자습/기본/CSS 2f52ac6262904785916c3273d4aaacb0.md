# CSS

마크업언어 (ex: html)이 몸체라면 CSS는 옷을 입히는 언어이다.

같은 HTML 코드를 사용했더라고 해도 CSS를 사용하여 전혀 다른 분위기의 웹사이트를 만들 수 있다.
